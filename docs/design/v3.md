# MCPixel v3 UX

Ink-only local suite. No Forest theme, no theme switcher, no decorative corner glow.

## Layout

- **Left rail**: Queue | Projects tabs + **New**
- **Main pane modes**: empty state, Create workspace, or job inspector
- **New** opens Create on the right (prompt, size chips as prompt hints, generate, upload, advanced)
- Selecting a job returns the main pane to the inspector
- Mobile keeps Create | Queue | Job tabs aligned with the same model

## Visual

- Flat `var(--bg0)` black background
- Yellow CTA buttons, squared chrome, Pixelify Sans + IBM Plex Mono
- Stage strip shows **edited** only when an edit PNG exists

## Projects

Lightweight grouping in `data/projects.json`:

- id, name, job_ids[], created_at / updated_at
- API: list / create / rename / delete; add / remove job
- UI: project list → drill into jobs; **Unfiled** for jobs with no project
- Job menu: Add to project / Remove from project

## Settings

Top-bar Settings drawer:

- `OPENAI_API_KEY`, optional `REMOVE_BG_API_KEY`
- Read-only note that jobs live under `data/jobs`
- Persist via `GET/PUT /v1/settings` → `data/settings.json`
- Hot-apply on save; env values still win on process boot when already set
