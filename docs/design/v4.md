## API

| Endpoint | Role |
| --- | --- |
| `POST /v1/prompt/refine` | `{prompt}` → `{refined}` |
| `POST /v1/generate` | JSON generate; optional `reference_job_id` + `reference_stage` |
| `POST /v1/generate/with-reference` | Multipart prompt + `reference` file |

Reference PNGs are stored as `data/jobs/<id>/reference.png` when used.
